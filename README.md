# SEO London

## Before cloning
```bash
brew install python3
pip3 install --upgrade pip setuptools wheel
pip3 install virtualenv
mkdir seolondon
cd seolondon
virtualenv env
source env/bin/activate
pip3 install djangocms-installer
mkdir seolondon
cd seolondon
```

## Clone
```bash
git clone <>
```

## After cloning
```bash
npm install
gulp
npm run server (simultaneously with gulp)
```
